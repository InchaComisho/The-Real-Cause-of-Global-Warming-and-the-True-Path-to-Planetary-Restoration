## 概要

**温暖化の本当の原因と、地球再生への真の道**は、現代の地球温暖化を単に大気中CO₂濃度の増加だけで説明するのではなく、その背後にある地球の炭素固定システム、特に土壌微生物と腐植形成の崩壊から再解釈する論考である。

一般的には、地球温暖化は「CO₂が増えたから気温が上がった」と説明される。もちろん、CO₂削減は必要である。しかし本稿では、より根本的な因果関係を次のように整理する。

> 現代の温暖化の一次的な原因は、炭素排出そのものだけではない。  
> 土壌微生物の破壊によって、地球の自然な炭素固定システムが崩壊したことである。

戦後の大規模農業、化学肥料依存、農薬使用、単一栽培、単一植林、土壌圧縮は、土壌生態系の基盤を劣化させた。その結果、植物が吸収した炭素を地下に安定固定する仕組みが弱まり、大気中にCO₂が蓄積しやすくなった。

つまり、CO₂は温暖化の重要な加速要因であるが、根本原因のすべてではない。  
本当の問題は、地球の炭素固定源そのものが失われたことである。

## 要旨

本稿の中心命題は、次である。

```text
微生物の減少
→ 炭素固定力の低下
→ 大気中CO₂の蓄積
→ 気温上昇
→ 土壌乾燥・微生物減少の加速
→ さらなる炭素固定力の低下
```

この構造では、CO₂削減だけでは不十分である。CO₂排出を減らしても、土壌・森林・海洋・微生物による炭素吸収システムが壊れたままでは、地球は回復しない。

そのため、地球再生には二層の戦略が必要である。

1. **長期修復：土壌微生物と炭素固定システムの回復**
2. **短期安定化：ミスト冷却・水循環補完による生命圏の保護**

これは、医療にたとえるなら、次の関係である。

- CO₂削減 = 病気の進行を遅らせる処置
- ミスト冷却 = 患者の容体を安定させる応急処置
- 土壌微生物回復 = 根本治療

## 1. 戦後の生態系転換

第二次世界大戦後、世界の農業と林業は、収量と工業的生産性を最大化する方向へ急速に変化した。

### 戦後に広がった変化

- 単一作物栽培
- 均一な人工林・単一植林
- 化学肥料の大量使用
- 広域農薬の散布
- 機械化による土壌圧縮
- 深耕・反復耕起
- 有機物を土へ戻さない農業構造

これらは短期的な生産性を高めた一方で、土壌の生命基盤を破壊した。

土壌は単なる鉱物の集合ではない。土壌は、微生物、菌類、根、ミミズ、昆虫、有機物、水、空気が相互作用する生きた構造である。

この生きた構造が弱まったことで、地球の炭素固定装置が機能不全に陥った。

## 2. 土壌はかつて地球の炭素エンジンだった

健全な生態系では、炭素は次のように循環していた。

```text
大気中CO₂
→ 植物の光合成
→ 根からの分泌物
→ 微生物による処理
→ 腐植形成
→ 安定した土壌炭素
```

この仕組みは、単に植物がCO₂を吸うだけでは成立しない。

植物が吸収した炭素は、根を通じて土壌微生物へ渡され、微生物と菌類によって安定した有機炭素、すなわち腐植として土壌に蓄えられる。

### 健全な土壌炭素システムの役割

- 炭素を地下に長期保存する。
- 土壌の保水力を高める。
- 植物の根を支える。
- 昆虫や土壌生物を支える。
- 水循環を安定させる。
- 地表温度の上昇を抑える。
- 気候安定性に寄与する。

つまり、土壌は地球の炭素エンジンだった。

## 3. 化学農業が炭素固定ループを壊した

化学肥料は、植物に直接栄養を与える。

一見すると効率的である。しかし、この構造は土壌微生物を不要なものとして扱う。

植物は成長しても、土壌微生物との共生、腐植形成、炭素の長期固定が弱まる。農薬は昆虫だけでなく、土壌中の共生菌、細菌、根圏生態系にも影響を与える。

### 化学農業による結果

- 微生物群集の減少
- 腐植生成の低下
- 土壌保水力の低下
- 土壌の硬化・鉱物化
- 炭素固定の停滞
- 乾燥と高温への脆弱化
- 栄養塩流出
- 河川・海洋の富栄養化
- 海洋酸欠域の拡大

このとき、植物による炭素吸収は一部続いていても、炭素の安定貯蔵が崩れる。

```text
炭素吸収は続いた
しかし炭素貯蔵が崩壊した
その結果、大気中CO₂の蓄積が加速した
```

## 4. 気候の時間差

生態系の破壊は、すぐに地表気温へ現れるわけではない。

地球規模の反応には、20〜50年程度の時間差が生じることがある。

### 因果の時間差

```text
1940〜1950年代：戦後農業・化学肥料・農薬・単一植林の拡大
↓
数十年かけて土壌微生物と腐植形成が低下
↓
炭素固定源が弱まる
↓
大気中CO₂が蓄積しやすくなる
↓
1970年前後から全球的な気温異常として表面化
```

この時間差が、「CO₂が増えたから温暖化した」という単純な説明を生みやすくした。

しかし本稿では、次の因果連鎖を重視する。

```text
微生物の減少
→ 炭素固定源の喪失
→ CO₂蓄積
→ 気温上昇
```

CO₂は重要な加速要因である。  
しかし、最初の火種は炭素固定源の喪失だった可能性がある。

## 5. CO₂だけを見る対策が不十分な理由

CO₂削減は必要である。これは否定しない。

しかし、CO₂削減だけでは不十分である。

CO₂削減は、大気へ流入する炭素の量を減らす対策である。一方で、土壌や森林や海洋が炭素を吸収・固定する力が壊れたままでは、地球の代謝は回復しない。

### 土壌システムを回復しない場合

- CO₂削減は増加速度を遅らせるだけになる。
- 地球の熱慣性により、気温上昇は続く。
- 高温と乾燥により土壌微生物はさらに減少する。
- 炭素吸収力はさらに低下する。
- 温暖化フィードバックが強まる。

### フィードバックの罠

```text
高温化
→ 土壌乾燥
→ 微生物減少
→ 炭素吸収力低下
→ CO₂増加
→ さらなる高温化
```

したがって、排出削減と同時に、炭素固定源の回復が必要である。
